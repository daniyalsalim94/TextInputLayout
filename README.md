## Setup

Add it in your root build.gradle at the end of repositories:
```groovy
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

Add the dependency:

```groovy
dependencies {
	        implementation 'com.github.daniyalsalim94:TextInputLayout:1.0'
	}
```
